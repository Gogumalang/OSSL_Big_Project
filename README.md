## 기능
### 1) 기본 프로그램
 - 로그인 [ 서버 관리자 / 클라이언트 ] 
- 해당 클라이언트의 잔고 리턴



### 2) 서버 관리자

 - 선수 등록
 - 선수 능력치 수정
 - 선수 삭제
 - 이적시장 선수 정보 조회
 - 저장 및 종료

### 3) 클라이언트
 
  - 선수 구매
  - 선수 판매
  - 구입한 선수 목록 조회
  - 선수 비교
  - 저장 및 종료
